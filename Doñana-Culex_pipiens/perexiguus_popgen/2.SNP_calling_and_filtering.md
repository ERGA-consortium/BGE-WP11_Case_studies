### SNP filtering
#### 1. highly repeated regions and callable sites
In the case of Culex perexiguus, the reference genome has a hard-masked version available here https://ftp.ebi.ac.uk/pub/ensemblorganisms/Culex_perexiguus/GCA_964243045.1/genome/ ,so it's not necessary to mask highly repeated regions beforehand.

To obtain bed files with repeats from hard masked reference genome i used the script in https://www.danielecook.com/generate-a-bedfile-of-masked-ranges-a-fasta-file/

Then I split the bed file by chromosome using awk:
```bash

awk '$1=="1"' perex_hardmasked_repeats.bed > c1_perex_repeats.bed
awk '$1=="2"' perex_hardmasked_repeats.bed > c2_perex_repeats.bed
awk '$1=="3"' perex_hardmasked_repeats.bed > c3_perex_repeats.bed
```

and did the filter of highly repeated regions from each chromosome
```bash
bcftools filter -M $repeat_bed --soft-filter REPMASK --threads 15 -Ob -o $repeat_masked_bcf $input_bcf
```

The I used [filter-callable-sites-perex.sh](../scripts/used_4_perex/filter-callable-sites-perex.sh) to filter and count how many callable sites are there in each chromsomes (minimum allele count=1; varian and invariant sites).
Than script also extract data on misisng sites and coverage depth to assess dataset's quality.

#### 2. SNP filtering

Filters applied:
* retain only biallellic SNPs
* exclude sirtes with INFO/DP> meanDP*2
* DP per site & individual >=5
* GQ per site & individual >=30
* Missinf GT per site < 50% (for a first check)

This filters are still aplied sepparately for each chromosome. I used [filter-variants.sh](../scripts/filter-variants.sh). The scripts also extracts missing data statistics after each filtering step.

Missing sites per individual were plotted using [plot-missingness.R](../scripts/plot_missingess.R)
in order to select samples that should be excluded due to high amount of missing sites. 

Before plotting missing data, is important to sort, index and merge the bcf files from the 3 chromosomes in order to account for total missing sites.

```bash
for bcf in *BDQM5.bcf; do
echo "sorting $bcf"
name="${bcf%.bcf}"
output="${name}_sorted.bcf"
echo "Output file: $output"
bcftools sort --temp-dir /sietch_colab/scamison/talapas/variant_filtering/temp-bcftools -Ob -o $output $bcf
echo "done with $bcf"
done
 #also indexed them (show just 1 as example)
for bcf in *BDQM5_sorted.bcf; do
echo "indexing $bcf"
bcftools index $bcf
done

#then I concat them
bcftools concat -Ob --threads 30 -o 530perex_perex.bcf 53023perex_1_BDQM5_sorted.bcf 53023perex_2_BDQM5_sorted.bcf 53023perex_3_BDQM5_sorted.bcf
#index concat
bcftools index 530perex_perex.bcf
#the recalculate the missingness per individual
vcftools --bcf  530perex_perex.bcf --out 530perex_perex --missing-indv
bcftools +counts
```
Then i exclude samples with high levels of missing genotypes using _bcftools view -s $samples_ 

FInally, I apply last filters:
* retain snps with <20% missing genotypes
* retain alleles with MAF >= 0.5 para comparar

I used  [filter_variants_MISyMAF5.sh](../scripts/used_4_perex/filter_variants_MISyMAF5.sh).
