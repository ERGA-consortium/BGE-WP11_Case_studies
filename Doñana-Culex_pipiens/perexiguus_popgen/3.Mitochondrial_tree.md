To confirm all anallyzed samples were Culex perexiguus we did a phylogenetic reconstruction using mitochondrial DNA.

### 1.obtain MT consensus
First, I extracted MT DNA. Only primary alignments with MAPQ ≥30
```bash
for bam in *F8.filter60.bam; do
  echo "Processing $bam"
  out_bam=${bam%_F8.filter60.bam}
  samtools view -@ 6 -h -b -F 0x900 -q 30 "$bam" MT | \
    samtools sort -@ 6 -o "for-consensus/${out_bam}_clean.bam" -
  samtools index -@ 6 "for-consensus/${out_bam}_clean.bam"
done
```
Then I run [consensus-mitoc-perex-.sh]. This script obtain consensus sequences for MT DNA considering haploidy and masking sites with low depth coverage.

### 2. Alignment and pylogenetic reconstruction
I corrected sequences' names and them int a multi-sequence fasta file.
```bash
```bash
# In the folder with the PX*.fasta files:
sed -E -i 's/^>(PX[0-9]+).*/>\1/' *.fasta
# Now concatenate all the fasta files into one
cat ./data/perexiguus/*_consensus.fasta \
    ./*fasta ./*fa > mitoc_seq_perexiguus.fasta
```

This fasta files already include sequences from reference genomes from other species and Culex perexiguus.

* GCA_964213205.1	idCulMode1.1 assembly for Culex modestus
* GCA_000209185.1	CulPip1.0 assembly for Culex quinquefasciatus
* GCA_964187845.1	idCulLati1.1 assembly for Culex laticinctus
* GCA_964340515.1	idCulThei1.1 assembly for * Culex theileri
* GCA_963924435.1	idCulPipi1.1 assembly for Culex pipiens
* GCA_964243045.1	idCulPerx1.1 assembly for Culex perexiguus


Then I aligned the consensus and reference sequences and run the tree
```bash
#alignment
mafft --auto "$INPUT" > "$OUTPUT"
#trees
iqtree -s  "$INPUT" -m MFP -b 1000 -nt 8 -seed 12345 -pre "$OUTPUT"
#consensus tree
iqtree2 -con -t "$INPUT" -pre "$OUTPUT"
```

### 3. Phylogenetic reconstruction with Culex univitattus samples
In order to confirm all my samples were Culex pipiens and not Culex univitattus, I redid mitochondrial phylogenetic reconstruction based on the marker COI

#### 2.1: download Cx univitattus sequences

I installed entrez-direct and searched for the sequences

```bash
# Search IDs
esearch -db nucleotide -query '"Culex univittatus"[Organism] AND (COI OR CO1 OR COX1 OR cox1 OR coxI)' \
  | efetch -format acc > univitattus_COI.ids

# Download all sequences in FASTA
esearch -db nucleotide -query '"Culex univittatus"[Organism] AND (COI OR CO1 OR COX1 OR cox1 OR coxI)' \
  | efetch -format fasta > univitattus_COI.fasta


# Download full GenBank records (with metadata: country, collection_date, isolation source…)
# Add header row
printf "Accession\tTitle\tOrganism\tLength\tcountry\tcollection_date\tlat_lon\tisolate\thost\tisolation_source\n" \
  > univitattus_COI_metadata.tsv
```
I aligned all sequences (Cx. univitattus COI sequences and my sample MT.DNA) using Geneious v2023.0.4 to do a quick search of COI marker. Once located the approximate coordinates (pb 4162 y 4904) y cut Cx. perexiguus samples around that region and exported all sequences (not the alignment) to a multi-sequence fasta file.  Then I aligned them and reconstructed the phylogeny using the same mafft and iqtree2 commands as before.
