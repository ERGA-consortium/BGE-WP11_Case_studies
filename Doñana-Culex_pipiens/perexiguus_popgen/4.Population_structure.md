### 1: wide-genome PCA
I run LD prunning (--indep-pairwise 200 20 .02) PCA using PLINK.

First get only GT data from bcf files
```bash
bcftools annotate -x ^FORMAT/GT 53025perex_all_clean_BDQMF.bcf  -Oz -o perex_all_filteredGT.vcf.gz
tabix -p vcf perex_all_filteredGT.vcf.gz
```
Then run [ld-prunning2.sh](../scripts/ld-prunning2.sh)
(that script was used for pipiens analyses so i just changed --indep-pairwise parameter when needed).

### 2. window PCA
In order to check genetic variation along eachof the chromosomes I use winPCA (https://github.com/MoritzBlumer/winpca?tab=readme-ov-file)

```bash
./winpca/winpca pca data/perexiguus/perex_all_filteredGT.vcf.gz 1:1-165885053 perex_1_wpca -m 0.05 -i 100000 ;
./winpca/winpca pca data/perexiguus/perex_all_filteredGT.vcf.gz 2:1-150416927 perex_2_wpca -m 0.05 -i 100000 ;
./winpca/winpca pca data/perexiguus/perex_all_filteredGT.vcf.gz 3:1-109403336 perex_3_wpca -m 0.05 -i 100000
#and obtain plots
#to plot the already generated PCA
./winpca/winpca chromplot perex_1_wpca 1:1-165885053 -m data/perexiguus/meta-perex-143samples.txt -g location ;
./winpca/winpca chromplot perex_2_wpca 2:1-150416927 -m data/perexiguus/meta-perex-143samples.txt -g location ;
./winpca/winpca chromplot perex_3_wpca 3:1-109403336 -m data/perexiguus/meta-perex-143samples.txt -g location 
```
