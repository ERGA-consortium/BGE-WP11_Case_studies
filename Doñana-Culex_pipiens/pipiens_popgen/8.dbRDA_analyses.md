# Population genomics of Culex pipiens from Southwestern Spain
## Part 7: Characterization of WOlbachia infection
### Sonia Cebrian-Camison 

[distance_base_redundancy.R](../scripts/distance_base_redundancy.R) contains the code used to run dbRDA analyses to test the association of genetic distances with geographic location, habitat type and infection by Wolbachia. dbRDA is run using the *vegan* package in R. Genetic distances are calculated based on PCA results. Geographic distances are represented by PCNM variables calculated from geographic coordinates (pcnm() in *vegan*). Habitat type and Wolbachia infection are included as factors. Significance of the models and terms is tested using ANOVA-like permutation tests.

This script also contains the code to run mantel test to check isolation by ditance for Culex pipiens pipiens and Culex pipiens molestus sepparetely. Finally, it runs a bete GLMM to model the effect of habitat (urban/rural) onver individual ancestry proportions.
