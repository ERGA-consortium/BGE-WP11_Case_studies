# Spongenes: adaptation genomics of the endangered sponge Spongia officinalis through whole genome (re)sequencing
This repository includes all the scripts used to analyze (from raw data to SNP calling) the whole genome re-sequencing data of the BGE case study focused on Spongia officinalis.
